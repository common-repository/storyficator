Storyficator" is a dynamic WordPress plugin designed to enhance user engagement and storytelling on websites.

== Description ==

Storyficator provides a visually captivating experience, allowing website visitors to swipe or click through a series of bite-sized stories presented in a carousel format. == Frequently Asked Questions ==

= How do I install Storyficator? =

There are two ways you can install Storyficator plugin.

You can install the plugin from your WordPress dashboard:
1. Navigate to your WordPress Dashboard -> Plugins -> Add New
2. Search for Storyficator plugin
3. Install and Activate Storyficator from your Plugins page.

Or you can install it directly from WordPress.org
1. Locate and download Storyficator on wordpress.org
2. Upload the storyficator directory to your ‘/wp-content/plugins/’ directory using your preferred method (ftp, sftp, scp, etc…)
3. Install and Activate Storyficator plugin from your Plugins page.


= Does Storyficator plugin work with any WordPress theme? =

Yes.
